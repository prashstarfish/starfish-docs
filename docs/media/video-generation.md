# Video Generation

Create videos from text descriptions or images within Starfish. Configure duration, aspect ratio, and starting frame to match your needs.


## Configuration Options

| Option | Values | Default |
|--------|--------|---------|
| **Duration** | 3 to 60 seconds | -- |
| **Aspect ratio** | 16:9 (landscape), 9:16 (portrait), 1:1 (square) | 16:9 |
| **Start frame** | Optional image upload | None |

## How to Use

1. **Create a session**: Click the Video Generation option in the sidebar
2. **Configure settings**: Set duration and aspect ratio for your target format
3. **Add a start frame** *(optional)*: Upload an image to use as the first frame
4. **Describe the video**: Write a natural language description of the video you want
5. **Generate**: Click generate and wait for processing to complete


Video generation takes longer than image generation. A progress indicator shows the current status while you wait.

## Preview and Download

Once generation completes:

- **Preview** the video directly in Starfish
- **Download** to your local filesystem

## Aspect Ratio Guide

Choose the right aspect ratio for your platform:

| Aspect Ratio | Format | Common Uses |
|--------------|--------|-------------|
| 16:9 | Landscape | YouTube, presentations, websites |
| 9:16 | Portrait | Instagram Stories, TikTok, YouTube Shorts |
| 1:1 | Square | Instagram feed, social ads |

## Use Cases

- **Social media content**: Generate short-form videos for Stories, Reels, and TikTok
- **Product demos**: Create quick product walkthroughs and feature highlights
- **Ads**: Produce video ad creatives for testing and iteration
- **Presentations**: Add dynamic video content to slide decks
