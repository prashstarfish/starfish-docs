# Integrations Overview

Starfish connects to the tools your team already uses. Each integration runs as a local MCP server, giving the AI workspace direct access to read and write data in your connected apps.


## All Integrations

| Name | Category | Auth Type | What It Does |
| --- | --- | --- | --- |
| [ClickUp](./clickup) | Productivity | Direct OAuth | Manage tasks, projects, docs, and time tracking |
| [Google Workspace](./google-workspace) | Productivity | Stream / CLI setup | Gmail, Calendar, Drive, Docs, Sheets, Slides, Tasks, Chat, Contacts |
| [Zapier](./zapier) | Productivity | OAuth PKCE | Trigger Zaps and automate across thousands of apps |
| [GoHighLevel](./gohighlevel) | Productivity | Personal Access Token | CRM contacts, pipelines, conversations, and social media |
| [Figma](./figma) | Design | OAuth PKCE | Read designs, inspect components, generate code |
| [Canva](./canva) | Design | OAuth PKCE | Create and edit designs, presentations, and marketing materials |
| [Stitch](./stitch) | Design | Personal Access Token | Generate UI screens, design systems, and app prototypes |
| [GitHub](./github) | Development | Stream / CLI setup | Repos, pull requests, issues, and code reviews |
| [Vercel](./vercel) | Development | OAuth PKCE | Deploy projects, check builds, manage domains |
| [Playwright](./playwright) | Development | None (local) | Browser automation, screenshots, and page interaction |
| [Google Marketing Suite](./google-marketing) | Data & Analytics | Stream / CLI setup | GA4 analytics, Search Console, Tag Manager |
| [Facebook Ads](./facebook-ads) | Data & Analytics | Team-proxy / PAT fallback | Ad campaigns, audiences, creatives, and performance insights |
| [Apify](./apify) | Data & Analytics | OAuth PKCE | Web scraping, data extraction, and automation actors |

## By Category

### Productivity

Manage your work, automate workflows, and keep your CRM in sync.

- **[ClickUp](./clickup)** -- Tasks, projects, docs, time tracking
- **[Google Workspace](./google-workspace)** -- Gmail, Calendar, Drive, Docs, Sheets, Slides, Tasks, Chat, Contacts
- **[Zapier](./zapier)** -- Trigger Zaps across thousands of apps
- **[GoHighLevel](./gohighlevel)** -- CRM, pipelines, campaigns, social media

### Design

Create, inspect, and iterate on designs without leaving Starfish.

- **[Figma](./figma)** -- Read designs, generate code, Code Connect
- **[Canva](./canva)** -- Designs, presentations, marketing materials
- **[Stitch](./stitch)** -- UI screens, design systems, app prototypes

### Development

Ship code, deploy, and test from your AI workspace.

- **[GitHub](./github)** -- Repos, PRs, issues, code reviews
- **[Vercel](./vercel)** -- Deployments, build logs, domains
- **[Playwright](./playwright)** -- Browser automation and testing

### Data & Analytics

Pull performance data, scrape the web, and analyze results.

- **[Google Marketing Suite](./google-marketing)** -- GA4, Search Console, Tag Manager
- **[Facebook Ads](./facebook-ads)** -- Campaigns, audiences, creatives, insights
- **[Apify](./apify)** -- Web scraping and data extraction at scale
