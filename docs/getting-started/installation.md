# Installation

Starfish is currently available for macOS on Apple Silicon.

## System requirements

| Requirement | Minimum |
| --- | --- |
| OS | macOS 13 (Ventura) or later |
| Chip | Apple Silicon (M1, M2, M3, M4) |
| RAM | 4 GB |

## Download

Download the latest `.dmg` from the [GitHub Releases](https://github.com/starfish-solutions/starfish-app/releases) page or from the Starfish website.

## Install

1. Open the downloaded `.dmg` file.
2. Drag **Starfish** into your **Applications** folder.
3. Eject the disk image.


## First launch

macOS may block the app because it was downloaded from the internet. If you see a warning:

1. Open **Finder** and navigate to **Applications**.
2. Right-click (or Control-click) on **Starfish**.
3. Select **Open** from the context menu.
4. Click **Open** in the confirmation dialog.

You only need to do this once. Subsequent launches work normally from the Dock or Spotlight.


## What's next

After launching for the first time, the [Setup Wizard](/getting-started/setup-wizard) walks you through account creation, plan selection, and connecting your first integration.
